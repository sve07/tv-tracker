Settings

TMDB API Access Token

Required to fetch series data from TMDB. Get a free "API Read Access Token" (not the shorter v3 API key) at themoviedb.org/settings/api. Stored locally in this browser only — never sent anywhere except TMDB itself.

Appearance

Notifications

In-app notifications work whenever the app is open. Background notifications (via periodic background sync) are best-effort only — supported on Chromium browsers with the app installed as a PWA, and not guaranteed to fire while the app is fully closed.
